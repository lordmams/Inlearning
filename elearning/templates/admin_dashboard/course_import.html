Guide d'importation
Format CSV

Format simple avec en-têtes. Idéal pour Excel et Google Sheets.

Champs requis: title, description, category, instructor

Format JSON

Format structuré permettant les leçons et métadonnées avancées.

Avantages: Leçons, tags, structure complexe

Format Excel

Fichiers .xlsx avec support des formules et mise en forme.

Limites: Structure simple comme CSV

  4. Indexation - Ajout à Elasticsearch pour la recherche
  5. Notification - Rapport de résultats
